Place the Kaggle CSV here after download:

- Expected default file: `BangaloreHouseRentDtls.csv` (schema: `Locality`, `MinPrice`, `MaxPrice`, `AvgRent`, `HouseType`)
- Override path: set env `BLR_RAW_CSV` before running `ingest/normalize_and_enrich.py`
- Legacy filename `bangalore_rent.csv` also works if you rename or symlink
- Dataset: https://www.kaggle.com/datasets/csunnikrishnan/bangalore-house-rent-details-1bhk-2bhk-3bhk

CSV files in this folder are gitignored.
